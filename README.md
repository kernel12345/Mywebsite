# 兄弟CS网站

## 项目简介

这是一个基于 Vue 3 + Vite + Express + MongoDB 的全栈网站项目，专为兄弟CS社区打造。

## 功能特性

### 前端功能
- **响应式设计**：适配不同屏幕尺寸，提供良好的用户体验
- **实时时钟**：显示当前时间和日期
- **随机名言**：每10秒自动更新一句随机名言，带有淡入淡出效果
- **天气数据**：显示当前城市、温度和天气描述
- **随机背景图片**：使用二次元图片 API 获取随机背景图片
- **社交图标**：包含 GitHub、微信等社交平台链接，带有悬停效果
- **视觉特效**：添加雪花飘落和鼠标点击特效
- **自定义鼠标指针**：使用自定义鼠标指针图片
- **浏览器标题图片**：使用自定义图标作为浏览器标题图片

### 核心功能模块
- **博客系统**：包含文章列表、分类标签、热门标签等
- **用户系统**：实现了用户注册和登录功能，内置管理员账号
- **小店系统**：完整的商品展示、分类标签、搜索功能等
- **管理后台**：
  - 服务器状态监控
  - 应用状态管理
  - 访问统计和用户数量显示
  - 文章管理（添加、编辑、删除、状态控制）
  - 商品管理（上架、下架、添加、编辑）

## 技术栈

### 前端
- **框架**：Vue 3
- **构建工具**：Vite
- **路由**：Vue Router
- **样式**：CSS3，包含毛玻璃效果

### 后端
- **框架**：Express
- **数据库**：MongoDB
- **ORM**：Mongoose

## 安装与运行

### 前置要求
- Node.js 14.0+
- MongoDB 4.0+

### 前端安装与运行

```bash
# 安装依赖
npm install

# 启动开发服务器
npm run dev

# 构建生产版本
npm run build

# 预览生产版本
npm run preview
```

### 后端安装与运行

```bash
# 进入后端目录
cd backend

# 安装依赖
npm install

# 启动后端服务器
npm start

# 启动开发模式（带热重载）
npm run dev
```
